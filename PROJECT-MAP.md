## 6. HOW THINGS CONNECT

```
You edit the blog post wording here:
  _posts/2026-03-05-staggered-did-event-study.md

Charts are generated by running:
  python assets/data/staggered_did_simulation.py
  (outputs to assets/images/posts/staggered-did/)

Math renders via MathJax configured in:
  _includes/head/custom.html
  (MUST keep markdown="1" on all <details> tags in the post)

The post shows up on the works page because of:
  categories: [applied]  ← in the front matter
  _pages/works.md        ← filters by category

Author sidebar comes from:
  _config.yml → author.bio

"Other Works" heading comes from:
  _data/ui-text.yml

To preview locally:
  bundle exec jekyll serve
```
