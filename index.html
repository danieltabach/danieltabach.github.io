Data Scientist

I'm Danny, a Data Scientist at JPMorgan Chase. I work on large-scale optimization platforms, but I'm really a full-stack data scientist: messy data to creative solutions to production models to executive presentations. I like the whole arc.

I think of data science as part art, part science. The same problem usually has a dozen solutions, and I enjoy finding the one that actually fits. Currently finishing my MS in Analytics at Georgia Tech.

Build

I'm hands-on keyboard. Optimization models, ML pipelines, clean Python and SQL. I actually enjoy the building part.

Communicate

Every analysis needs a clear narrative and a "so what." I build roadmaps, work with cross-functional teams, and make technical work land.
